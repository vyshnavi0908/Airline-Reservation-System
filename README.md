# Airline Reservation System

This project is a database design for an Airline Reservation System using SQL.

## Features
- Passenger management
- Flight scheduling
- Ticket booking
- Payment processing
- Seat allocation
- Baggage tracking
- Loyalty programs
- Flight crew management

## Technologies Used
- SQL
- MySQL

## Database Tables
Passenger, Flight, Airport, Airline, Booking, Ticket, Seat, Payment, Baggage, Crew, Schedule, Promotion, LoyaltyProgram, CheckIn, Cancellation.
